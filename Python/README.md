# CodeWars - Python

Python solutions for Codewars katas, organized by difficulty.

## Structure

- `8 Kyu` up to `2 Kyu`.
- `dunno`: uncategorized solutions.

## Current snapshot

- Around 601 `.py` files.
- A few `.js` files stored alongside Python solutions.

## Usage

Run files directly with Python as needed:

```bash
python3 path/to/file.py
```
